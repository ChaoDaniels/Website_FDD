SCSS is een uitbreiding van SASS, en is de meest gebruikte versie van SASS. Het voordeel van SASS over normale CSS is dat SASS meer mogelijkheden biedt, zoals het gebruik van variabelen, nesting en herbruikbare stukjes code. Dit maakt SCSS erg fijn voor het organiseren van je CSS en het schrijven van minder code.

SCSS is een preprocessor, wat betekent dat het niet direct door de browser wordt gelezen. In plaats daarvan wordt de SCSS code omgezet naar CSS, wat de browser wel kan lezen. Dit gebeurt met behulp van een compiler, zoals bijvoorbeeld de SASS compiler.

Ik heb voor deze website al mijn CSS bestanden omgezet in SCSS en inmiddels veel gebruik gemaakt van de voordelen die SCSS biedt. Mijn SCSS is een stuk overzichtelijker.

Mijn gebruik van Frameworks en SCSS

Zelf heb ik geen gebruik gemaakt van Frameworks, omdat ik er weinig voordeel in zag toen het werd behandeld en nu gebruik ik veel SCSS. SCSS en een CSS Framework hebben allebei het doel om de nodige CSS te verminderen, maar omdat we een aparte les hadden waarin SCSS werd uitgelegd besloot ik hiervan gebruik te maken.
