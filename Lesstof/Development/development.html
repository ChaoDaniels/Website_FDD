HTML

In de eerste les van FDD hebben we geleerd wat een persona is en waarom het belangrijk is. Een persona is een soort template met info over een fictief persoon.

CSS

In de tweede week van FDD leerden we wat een sitemap is. Simpelgezegd, een sitemap is bijna letterlijk een kaart van hoe je de site moet navigeren. De map begint bij de pagina waar je het eerst op stuit, de “Homepage”. Op de meeste websites kan je vanuit deze pagina navigeren naar andere pagina’s.

Javascript

Deze pagina behandelt de basisprincipes van JavaScript, een programmeertaal die veel wordt gebruikt om interactieve elementen aan websites toe te voegen.

Git

Op deze pagina leer je over Git, een versiebeheersysteem dat wordt gebruikt om wijzigingen in broncode bij te houden tijdens softwareontwikkeling.

Frameworks & SCSS

Deze pagina geeft een overzicht van verschillende frameworks en SCSS, een uitbreiding op CSS die het schrijven van stijlen efficiënter maakt.

Responsive Web Design

Op deze pagina leer je over Responsive Web Design, een benadering om websites te maken die goed werken op verschillende apparaten en schermformaten.

SEO

Deze pagina behandelt SEO (Search Engine Optimization), technieken om de zichtbaarheid van een website in zoekmachines te verbeteren.
